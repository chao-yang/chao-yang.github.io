发表于 2016-05-05   |   分类于 Hexo   |     |   次阅读

这两天在Github上搭建了博客，以后就可以安心写博客啦~~期间遇到了一些问题，记录在此。

参考 zipperary 的这篇文章基本上就可以实现基于Github Page + Hexo 搭建博客。这里主要是补充下我在安装过程中遇到的问题。

主要问题

  1. 执行npm install报错 Error: ENOENT, chmod ‘/usr/local/lib/…’

    • 解决方法：这个错误是因为 npm 将目录下的.gitignore用作.npmignore文件，从而忽略了/lib。所以只需要在应用的根目录下增加一个空的.npmignore文件即可解决问题。
